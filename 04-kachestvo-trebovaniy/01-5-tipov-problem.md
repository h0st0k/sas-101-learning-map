# 5 типов проблем в требованиях

| Тип | Описание | Пример |
|-----|----------|--------|
| **Противоречия** | Два требования не могут быть выполнены одновременно | «Мгновенно» vs «пакетом раз в час» |
| **Пробелы** | Отсутствие информации для реализации | Описан основной сценарий, но не ошибочный |
| **Неоднозначности** | Формулировка понятна несколькими способами | «Система должна работать быстро» |
| **Дублирование** | Одно требование описано в двух местах | Повторяющиеся формулировки в разных разделах |
| **Избыточность** | Требования без ценности | «Система должна нравиться пользователям» |
