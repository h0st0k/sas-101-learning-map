# Postman — тестирование API

## Что такое Postman
Инструмент для отправки HTTP-запросов и тестирования API.

## Как использовать

### 1. Создать запрос
- Выбрать метод (GET, POST, PUT, DELETE)
- Ввести URL
- Добавить параметры (query, body, headers)
- Нажать Send

### 2. Работа с параметрами
- **Query-параметры** — добавляются в URL: ?type=square&position=center
- **Body** — JSON-данные для POST/PUT запросов
- **Headers** — заголовки (авторизация, Content-Type)

### 3. Анализ ответа
- **Status** — код ответа (200, 404, 500)
- **Body** — данные в JSON
- **Headers** — заголовки ответа

## Пример: GET /api/cats

**URL:** https://cataas.com/cat
**Метод:** GET
**Параметры:** type = square

**Ответ:** картинка кота

## Авторская приемка
- Аналитик проверяет API перед передачей в разработку
- Использует Postman для отправки запросов
- Проверяет корректность ответов
