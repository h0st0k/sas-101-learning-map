# Методы обследования

## 1. Интервью
- Открытые вопросы → дают контекст
- Закрытые вопросы → уточняют цифры
- Вопросы-триггеры → вскрывают боли
- Сценарные вопросы → моделируют будущее

## 2. Наблюдение (Shadowing)
- Аналитик ходит «тенью» за пользователем
- Фиксирует разрыв между «как должно быть» и «как есть»

## 3. Триангуляция
- Перекрёстная проверка из 3+ источников
- Интервью + Наблюдение + Логи = достоверность
