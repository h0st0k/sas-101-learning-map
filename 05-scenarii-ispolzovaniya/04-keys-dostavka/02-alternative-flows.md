# Кейс «Доставка продуктов» — Альтернативные сценарии

## А1. Шаг 4а — Оплата наличными
Клиент выбирает «Наличные при получении».  
Система показывает поле «Приготовить сдачу с».  
Клиент указывает сумму (или оставляет пустым).  
Кнопка меняется на «Заказать» — без немедленной оплаты.  
→ Возврат к шагу 6. Цель достигнута.

## А2. Шаг 5а — Самовывоз вместо доставки
Клиент выбирает «Самовывоз».  
Система показывает адреса пунктов выдачи и часы работы.  
Клиент выбирает пункт и время.  
Поле адреса доставки скрывается.  
→ Возврат к шагу 6. Цель достигнута.

## А3. Шаг 3а — Сохранённый адрес
У клиента уже есть сохранённый адрес в профиле.  
Система автоматически подставляет его в форму.  
Клиент не заполняет адрес вручную — только проверяет.  
→ Возврат к шагу 4. Цель достигнута.

## А4. Шаг 2а — Повтор прошлого заказа
Клиент нажимает «Повторить заказ» из истории.  
Система заполняет корзину теми же товарами и подставляет прошлый адрес и способ оплаты.  
Клиент проверяет данные и нажимает «Заказать».  
Шаги 3, 4, 5 пропускаются.  
→ Цель достигнута.
