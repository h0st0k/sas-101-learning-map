# Протокол встречи — «Прокачу»

**Дата:** 15.07.2026  
**Участники:** Руководитель стартапа, СТО (Дмитрий), Оператор поддержки (Анна), Системный аналитик  
**Цель:** Сбор требований к мобильному приложению для аренды велосипедов

## Ключевые тезисы

### Руководитель
- Хотим выйти из Telegram-бота в полноценное приложение
- Нужна карта с велосипедами, бронирование, оплата
- Планируем масштабироваться на 3 города (до 500 велосипедов)

### Дмитрий (СТО)
- Бот написан на коленке за 2 недели — не масштабируется
- Нужно мобильное приложение: карта, бронирование, оплата внутри
- Бюджет: 2 млн руб., срок: 3 месяца
- iOS и Android, данные карт — по стандарту PCI DSS

### Анна (Оператор поддержки)
- 132 обращения в месяц
- Главная боль — ручное завершение аренды (пользователи забывают)
- Приходится возвращать деньги вручную
- Нужно автозавершение по геолокации
- Не хватает карты станций и отчёта о поломках

## Договорённости
1. Разработать мобильное приложение (iOS + Android)
2. Реализовать карту с пинами 20 станций и статусами
3. Реализовать бронирование на 15 минут с push-уведомлением
4. Внедрить автозавершение аренды по геолокации
5. Интегрировать платёжный шлюз по PCI DSS
6. Добавить модуль отчёта о поломках и историю поездок
