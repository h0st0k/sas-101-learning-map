# UML — унифицированный язык моделирования

## Что такое UML
UML (Unified Modeling Language) — язык визуального моделирования для описания систем.

## Основные диаграммы UML

| Диаграмма | Для чего |
|-----------|----------|
| **Use Case диаграмма** | Показывает акторов и их цели в системе |
| **Диаграмма последовательности (Sequence)** | Взаимодействие объектов во времени |
| **Диаграмма классов** | Структура системы (классы, атрибуты, методы, связи) |
| **Диаграмма активности** | Алгоритмы и потоки управления |

## Use Case диаграмма
- Акторы (люди или системы)
- Use Cases (функции системы)
- Связи между ними

**Пример для библиотеки:**
- Актор: Читатель
- Use Cases: Поиск книги, Бронирование, Просмотр статуса
- Актор: Библиотекарь
- Use Cases: Управление книгами, Управление бронированиями

## Диаграмма последовательности (Sequence)
Показывает обмен сообщениями между объектами во времени.

### Элементы
- Линии жизни (объекты)
- Стрелки (сообщения)
- Фреймы (условия, циклы)

### Пример: «Читатель бронирует книгу»
1. Читатель → Система: Запрос на бронирование
2. Система → БД: Проверка доступности
3. БД → Система: Книга доступна
4. Система → Читатель: Подтверждение бронирования
