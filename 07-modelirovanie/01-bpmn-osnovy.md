# BPMN — моделирование бизнес-процессов

## Что такое BPMN
BPMN (Business Process Model and Notation) — нотация для описания бизнес-процессов. Позволяет визуализировать последовательность действий, участников и условия.

## Основные элементы

| Элемент | Обозначение | Назначение |
|---------|-------------|------------|
| **Пулы (Swimlanes)** | Дорожки | Участники процесса (роли, системы) |
| **Действие (Task)** | Прямоугольник | Конкретный шаг, выполняемый участником |
| **Событие (Event)** | Кружок | Начало (зелёный), промежуточное (жёлтый), конец (красный) |
| **Шлюз (Gateway)** | Ромб | Ветвление: XOR (только один путь), AND (все пути), OR (один или несколько) |
| **Поток (Sequence Flow)** | Стрелка | Переход между шагами |

## Пример BPMN: процесс «Запись к врачу»

| Пациент | Система |
|---------|---------|
| (Старт) → Открывает сайт → Выбирает специализацию → Выбирает врача → Выбирает время → Нажимает «Записаться» → (Конец) | → Показывает список врачей → Показывает расписание → Проверяет доступность → Создаёт запись → Отправляет уведомление |

## Шлюзы в BPMN

### XOR (Исключающее ИЛИ)
- Выбирается только один путь из нескольких
- Пример: если сумма > 100 000 → согласование, иначе — без согласования

### AND (Параллельное И)
- Запускаются все пути одновременно
- Пример: Отправить уведомление клиенту И начать сборку заказа
