# Инструменты системного аналитика

## 4 категории инструментов

| Категория | Инструменты | Для чего |
|-----------|-------------|----------|
| **Фиксация требований** | Jira, Trello, Яндекс.Трекер | User Stories, Use Cases, бэклог |
| **Моделирование** | Draw.io, PlantUML, Miro | BPMN, UML, ER-диаграммы |
| **Совместная работа** | Confluence, Яндекс.Вики, Figma | Документация, прототипы, обсуждения |
| **Ведение документации** | Confluence, Notion, Miro | Протоколы, архитектура, глоссарии |

## Jira — трекер задач
- Иерархия: Проект → EPIC → User Story → Задачи
- Канбан-доска: Backlog → In Progress → Done
- Спринты для Agile-команд
- Интеграция с Confluence

## Confluence — база знаний
- Единый источник правды
- Версионирование документов
- Интеграция с Jira (ссылки, макросы)
- Экспорт в Word/PDF

## Draw.io — редактор диаграмм
- Бесплатно, работает в браузере
- Поддерживает UML, BPMN 2.0, ERD
- Экспорт в PNG, SVG, XML
- Интеграция с Confluence, Google Drive

## Swagger/OpenAPI — документация API
- Описание REST-эндпоинтов
- Автогенерация UI
- Тестирование запросов прямо в браузере

## DBeaver — проводник по БД
- Подключение к любым БД (PostgreSQL, MySQL, и др.)
- Выполнение SQL-запросов
- Просмотр структуры таблиц
- Бесплатная версия
