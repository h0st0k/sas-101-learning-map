# Draw.io — редактор схем

## Для чего
- BPMN-диаграммы процессов
- UML-диаграммы
- ER-диаграммы (модели данных)
- Архитектурные схемы

## Основные элементы BPMN
- **Пулы** — участники процесса (дорожки)
- **Квадратики** — действия
- **Ромбы** — развилки (условия)
- **Кружки** — начало / конец процесса
- **Стрелки** — переходы между шагами

## Пример BPMN
Процесс «Запись к врачу»:
- Пул 1: Пациент (открывает сайт → выбирает врача → выбирает время → подтверждает)
- Пул 2: Система (показывает список → показывает расписание → создаёт запись → отправляет уведомление)

## Преимущества
- Бесплатно (браузер + десктоп)
- Интеграция с Confluence, Google Drive
- Экспорт в PNG, SVG, XML
