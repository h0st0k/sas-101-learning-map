# Miro — онлайн-доска для идей

## Для чего
- Мозговые штурмы
- Декомпозиция задач (стикеры → стрелочки)
- Канбан-доски
- Ретроспективы

## Как работает
- Бесконечный холст
- Стикеры, фигуры, стрелки
- Совместная работа в реальном времени
- Теги, лейблы, ссылки

## Пример использования
Сделать личный кабинет:
- Авторизация
- История заказов
- Личный профиль
- Текущие заказы

Каждая задача — отдельный стикер. Стрелки показывают зависимости. Галочка — задача выполнена.
