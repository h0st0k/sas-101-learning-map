# Jira — трекер задач

## Для чего
- Управление задачами (User Stories, баги, эпики)
- Планирование спринтов
- Отслеживание статусов
- Интеграция с Confluence и CI/CD

## Иерархия

## Основные поля
- **Резюме** — краткое описание
- **Статус** — To Do → In Progress → Done
- **Исполнитель** — кто делает
- **Приоритет** — Must / Should / Could / Won't
- **Метки** — для фильтрации
- **Родитель** — ссылка на EPIC

## Канбан-доска
- Backlog → In Progress → Ready for QA → Done
- Перетаскивание карточек между колонками
